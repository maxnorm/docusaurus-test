## Gradient Text

Use <GradientText>GradientText</GradientText> to make text stand out with beautiful gradients.

```jsx
<GradientText>Beautiful Gradient Text</GradientText>
<GradientText variant="secondary">Secondary Gradient</GradientText>
<GradientText variant="accent">Accent Gradient</GradientText>
```

Variants: <GradientText variant="primary">primary</GradientText>, <GradientText variant="secondary">secondary</GradientText>, <GradientText variant="accent">accent</GradientText>, <GradientText variant="success">success</GradientText>

## Usage in Documentation

All these components are available in your MDX files. Simply import them at the top of your document:

```jsx
import FeatureCard from '@site/src/components/FeatureCard';
import Badge from '@site/src/components/Badge';
import GradientText from '@site/src/components/GradientText';
// ... other imports
```

Then use them anywhere in your content!

<CalloutBox type="tip" title="Customization">
  All components support custom styling through className props and can be further customized by modifying their CSS modules.
</CalloutBox>
